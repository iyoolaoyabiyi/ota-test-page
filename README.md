# Onboarding Widget Test Page

A simple website designed for testing onboarding widget applications with multiple interactive elements.

## Overview

This test page provides a realistic website layout with 8 unique element IDs that can be used to test onboarding flows, tooltips, user guides, and product tours.

## Element IDs for Testing

The following element IDs are available for onboarding widget testing:

1. **`welcome-section`** - Header section with welcome message and tagline
2. **`navigation-menu`** - Main navigation bar with links
3. **`feature-showcase`** - Features section displaying key product features
4. **`cta-button`** - Primary call-to-action button ("Start Free Trial")
5. **`user-profile`** - User profile sidebar with avatar and user information
6. **`settings-panel`** - Settings section with toggle options
7. **`help-support`** - Help and support section with support buttons
8. **`footer-section`** - Footer with copyright and links

## Usage

1. Open `index.html` in your browser
2. Use the element IDs above to target specific elements in your onboarding widget
3. Test different onboarding flows and sequences

## Running Locally

Simply open the `index.html` file in any modern web browser, or serve it using a local web server:

```bash
# Using Python 3
python3 -m http.server 8080

# Using Node.js (with http-server package)
npx http-server -p 8080

# Then visit http://localhost:8080
```
